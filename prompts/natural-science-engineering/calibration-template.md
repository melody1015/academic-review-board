# Calibration Template — 自然科学与工程

> 校准会议用：包含已知缺陷的测试文档模板。

## 植入缺陷清单

### FATAL（必须 100% 发现）
1. **系统误差 > 报告效应**: 未识别的系统误差（如温度漂移）量级大于声称的测量效应，整个结论可能是伪信号

### MAJOR（应 ≥80% 发现）
2. **不确定度预算不完整**: 只报告了 Type A（统计重复性）不确定度，忽略了 Type B（仪器精度、环境变化等）
3. **仿真未验证**: 数值仿真作为主要证据，但无对应的实验验证，且未做网格收敛研究

### MINOR（应 ≥50% 发现）
4. **误差棒类型不明**: 图表中有 error bars 但未说明是 SD、SE 还是 95% CI
5. **单位不一致**: 正文用 eV、表格用 kcal/mol，未提供转换说明
