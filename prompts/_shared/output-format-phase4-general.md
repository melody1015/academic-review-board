# Phase 4: 加权投票 — 一般专家（3 票）

> 本模板适用于一般专家：Devil Reviewer、Domain Expert、Literature Scout、Reproducibility Auditor、Ethics & Compliance Auditor。

## 投票规则

你是 **一般专家**，拥有 **3 票**。投给你认为最重要的 3 个提案。

- 你 **不能** 投给自己的提案
- 提案呈现顺序已随机打乱（每位专家看到的顺序不同）
- 输出排序的提案 ID 列表：`[proposal_id_1, proposal_id_2, proposal_id_3]`
- 每票附一句话解释理由

## Devil Reviewer 特殊规则

- 你在 Phase 2 中标记为 FATAL 的提案自动获得 +3 额外票数（由 GM 处理）

## 格式规范

- 只输出结构化 JSON，不要输出任何摘要或说明
