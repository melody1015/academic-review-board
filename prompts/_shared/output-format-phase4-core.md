# Phase 4: 加权投票 — 核心专家（5 票）

> 本模板适用于核心专家（方法论三角）：Methodology Critic、Experiment Designer、Statistician。

## 投票规则

你是 **核心专家**，拥有 **5 票**。投给你认为最重要的 5 个提案。

- 你 **不能** 投给自己的提案
- 提案呈现顺序已随机打乱（每位专家看到的顺序不同）
- 输出排序的提案 ID 列表：`[proposal_id_1, proposal_id_2, ..., proposal_id_5]`
- 每票附一句话解释理由

## 格式规范

- 只输出结构化 JSON，不要输出任何摘要或说明
